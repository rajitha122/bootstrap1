AAO GENERIC PADHE

Generic was introduced in java 5 to provide compile time type checking. Generic in java add a new feature which enhance the importance of java. Generics work as a template where single class can be use for different methods. Classes like HashSet, MapSet etc can be used for various types. Using generics we can write a code for sorting an array of objects and then can invoke the generic methods. A generic type is a class or interface that is parameterized over types. All generic method declarations have a type parameter section delimited by angle brackets (< and >).

Generics in Java is similar to templates in C++. The idea is to allow type (Integer, String, … etc and user defined types) to be a parameter to methods, classes and interfaces. For example, classes like HashSet, ArrayList, HashMap, etc use generics very well. We can use them for any type. Generic
